Multivariate Analysis of Bird Morphological Data in R
Overview

This project applies multivariate statistical techniques to analyze bird morphology data using R. The analysis explores relationships among multiple biological measurements through exploratory data analysis, multivariate normality assessment, and dimensionality reduction techniques.

The project was completed as part of graduate coursework in Applied Statistics at the University of Oklahoma and demonstrates a reproducible statistical workflow using R Markdown.

Objectives
Explore the structure of multivariate biological data
Assess univariate and multivariate normality
Detect potential multivariate outliers using Mahalanobis distance
Visualize relationships among variables
Perform Principal Component Analysis (PCA)
Interpret multivariate patterns using statistical methods
Methods

The project includes:

Exploratory Data Analysis (EDA)
Summary statistics
Correlation analysis
Scatterplot matrices
Univariate normality assessment
Multivariate normality testing
Mahalanobis distance calculations
Chi-square Q-Q plots
Principal Component Analysis (PCA)
Data visualization using R
